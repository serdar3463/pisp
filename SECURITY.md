# Security Policy

## Supported Version

PISP Mobile 0.1.x is a beta-stage product.

## Current Security Controls

- Encrypted local vault.
- SecureStore-backed local encryption key.
- Device unlock before vault access.
- Background lock on app state change.
- Signed QR request and response payloads.
- Explicit consent before disclosure generation.
- Ledger entries contain hashes and metadata, not raw personal values.

## Required Before Public Launch

- Independent mobile security review.
- QR phishing and spoofing threat model.
- Key recovery and device loss policy.
- App Store and Google Play privacy review.
- Legal review for GDPR/KVKK.
