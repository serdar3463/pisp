# PISP Mağaza ve Kapalı Beta Kontrol Listesi

## TestFlight / İç Test Öncesi

- [ ] `npm run beta:check` temiz.
- [ ] `app.json` içindeki placeholder URL'ler değiştirildi.
- [ ] Final app icon ve splash görselleri eklendi.
- [ ] Fiziksel iPhone üzerinde test edildi.
- [ ] Fiziksel Android cihaz üzerinde test edildi.
- [ ] Kamera izin metni doğrulandı.
- [ ] Face ID / biyometrik kilit doğrulandı.
- [ ] Uygulama arka plana alındığında kilitlendiği doğrulandı.
- [ ] İki cihaz arasında QR request-response testi yapıldı.
- [ ] Yerel veri silme akışı doğrulandı.
- [ ] App Privacy etiketleri incelendi.
- [ ] `docs/QR_TWO_DEVICE_TEST_TR.md` başarıyla uygulandı.
- [ ] `docs/THREAT_MODEL_TR.md` kritik riskleri kapatacak şekilde güncellendi.

## Public Launch Öncesi

- [ ] `npm run launch:check` temiz.
- [ ] Gizlilik politikası hukuk incelemesi tamamlandı.
- [ ] Kullanım şartları hukuk incelemesi tamamlandı.
- [ ] KVKK/GDPR aydınlatma metni onaylandı.
- [ ] Veri silme süreci canlı çalışıyor.
- [ ] Destek URL'i canlı çalışıyor.
- [ ] Mobil güvenlik incelemesi tamamlandı.
- [ ] Kapalı beta geri bildirimleri çözüldü.
- [ ] Crash/error reporting kararı verildi.
- [ ] Production build imzalandı ve arşivlendi.
- [ ] `docs/V1_RELEASE_CHECKLIST_TR.md` içindeki kritik maddeler kapatıldı.
- [ ] `store/APP_STORE_METADATA_TR.md`, `store/GOOGLE_PLAY_METADATA_TR.md`, `store/PRIVACY_LABELS_TR.md` ve `store/REVIEW_NOTES_TR.md` mağaza hesabına işlendi.
