Veri Silme Talebi

Cihazındaki Veriler (Anında)

PISP kasandaki tüm veriler cihazında yerel olarak saklanır. Bunları anında silmek için:

1

Uygulamayı aç → Ayarlar sekmesine git

2

"Kasayı Sil" butonuna bas

3

Onay ver — tüm veriler kalıcı olarak silinir

Sunucu Tarafındaki Kayıtlar

Pazar yeri özelliğini kullandıysan cihaz hash'in ve token kayıtların sunucumuzda bulunabilir. Silme talebi için:

E-posta ile talep gönder:
serdarsimiek@gmail.com

Konu: "Veri Silme Talebi" — talebiniz 30 gün içinde işleme alınır.

Token Çekim Kayıtları

IBAN bilgisi çekim işleminden sonra sistemimizde tutulmaz. Token çekim talebi kaydının silinmesini istersen yukarıdaki e-posta adresinden bize ulaş.
