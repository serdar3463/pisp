# PISP Quality Gates

## Required For Every Release Candidate

- `npm run typecheck` passes.
- Two physical-device QR exchange test passes.
- Background lock test passes.
- Vault erase test passes.
- Explicit consent gate test passes.
- Invalid QR signature rejection test passes.
- Expired QR request rejection test passes.
- Privacy policy URL is live.
- Support URL is live.
- Data deletion URL is live.

## Required For Public Launch

- Independent legal review.
- Independent mobile security review.
- App Store privacy label review.
- Closed beta feedback triaged.
- Incident response contact configured.
- Data deletion support process tested.
