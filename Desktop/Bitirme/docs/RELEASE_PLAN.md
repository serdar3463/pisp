# PISP Release Plan

## v0.1 Internal Prototype

Goal: Validate core product flow.

- Local vault.
- Policy toggles.
- QR exchange.
- Proof ledger.
- Basic launch documentation.

## v0.2 Closed Beta

Goal: Test with 20-50 controlled users.

- Physical-device QR testing.
- Privacy copy review.
- Security center.
- Explicit consent review.
- Feedback collection.
- `npm run beta:check` gate.
- Store metadata draft.

## v0.3 Security Beta

Goal: Address production risk.

- External mobile security review.
- QR phishing/threat model improvements.
- Key recovery policy.
- Store privacy labels.
- Incident response and security disclosure process.

## v1.0 Public Launch

Goal: Public App Store / Google Play availability.

- Final legal documents.
- Final privacy/support/deletion URLs.
- Production builds.
- Store submission.
- `npm run launch:check` gate.
